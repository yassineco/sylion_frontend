# 01 – Vision & Positionnement SylionAI

## 1. Vision

SylionAI vise à devenir **la plateforme IA WhatsApp premium de référence au Maroc**, conçue pour automatiser :
- la qualification client,
- les réponses répétitives,
- la gestion des leads,
- les interactions multilingues (FR / AR / Darija / EN).

Philosophie : **Premium, robuste, rentable**.

## 2. Positionnement

- **Marché initial** : PME marocaines (cliniques, écoles, immobilier, cabinets…)
- **Différenciation** :
  - Assistant réellement multilingue dans la même conversation  
  - Architecture scalable (Gateway + Queue + Workers)  
  - RAG personnalisé sur documents internes  
  - Accompagnement complet pour 360dialog

## 3. Principes non négociables

1. Pas de dumping de prix : le produit est premium.  
2. Pas de promesses non tenues.  
3. Focus sur stabilité avant features gadgets.  
4. Inclure la sécurité (anti prompt injection, limites, garde-fous).

## 4. Objectifs 12 mois

- Atteindre le revenu pour dégager un salaire net cible de **30 000 MAD** + fond de roulement.  
- Obtenir **5–10 clients marocains premium**, puis **une offre internationale (LTD)**.  
- Stabiliser un produit industrialisable :  
  - RAG propre  
  - Admin console  
  - Multi-numéros WhatsApp  
  - Pipeline commercial complet
